# Entangle Chat

Quantum-inspired secure PWA for private text & voice chat. Twin AIs simulate entanglement to generate shared OTP pads via QR pairing. No phone number, no server logs, no metadata.

## Features
- Offline QR pairing
- E2EE text (OTP XOR)
- P2P voice (WebRTC + OTP)
- Offline-first (SW)
- Firebase signaling (free)

## Deploy
- Vercel: https://entangle-chat.vercel.app/
- GitHub Pages: Enable in Settings

## Setup
1. Pair via QR scan
2. Text: Type → Encrypt → Copy to partner
3. Voice: Create/Join call → Speak privately
