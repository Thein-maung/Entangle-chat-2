# Entangle Chat-2

Quantum-inspired secure PWA for private text & voice chat. Twin AIs simulate entanglement to generate shared OTP pads via QR pairing. No phone number, no server logs, no metadata.

## Features
- Offline QR pairing (entanglement)
- E2EE text (AI-generated OTP XOR)
- P2P voice (WebRTC + OTP chunks)
- Offline-first (SW cache)
- Firebase signaling (free)

## Deploy
- Vercel: https://entangle-chat-2.vercel.app/
- GitHub Pages: Enable in Settings

## Setup
1. Pair via QR scan (shared seed)
2. Text: Type → Encrypt → Copy to partner
3. Voice: Create/Join call → Speak privately (pads auto-sync)
